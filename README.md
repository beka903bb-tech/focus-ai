# Focus AI 🎯

Vaqtga asoslangan odatlar kuzatuvchisi — haqiqiy sarflangan vaqtni hisoblaydi.

## Texnologiyalar

- React Native + Expo SDK 54
- AsyncStorage (offline)
- DeepSeek V3 AI (murabbiy)
- React Navigation v6

## Ekranlar

- Onboarding, Auth, Dashboard, Session taymer, Statistika, AI Murabbiy, Profil

## Ishga tushirish

```bash
npm install
npx expo start
```
